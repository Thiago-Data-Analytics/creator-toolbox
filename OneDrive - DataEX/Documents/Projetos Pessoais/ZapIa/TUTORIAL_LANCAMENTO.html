Adicionar domínio no Cloudflare e configurar DNS
VOCÊ

1. Acesse dash.cloudflare.com

2. Clique em "+ Add a domain"

3. Digite mercabot.com.br → plano Free → Continue

4. Cloudflare vai mostrar os nameservers — confirme que são os mesmos que você configurou no registro.br

5. No Cloudflare → aba DNS → adicione este registro:

TipoNomeConteúdoProxy
CNAMEapiapi.mercabot.com.br✅ Proxied

Aguarde propagação (geralmente <1h, máximo 24h).

Configurar Cloudflare Access no painel-parceiro
VOCÊ

1. No Cloudflare → menu lateral → Zero Trust

2. Access → Applications → "+ Add an application"

3. Escolha "Self-hosted"

4. Preencha:

  • Application name: MercaBot Painel Parceiro
  • Application domain: mercabot.com.br/painel-parceiro
  • Session duration: 24 hours

5. Em Policies → Add Policy:

  • Nome: Acesso restrito
  • Action: Allow
  • Rule: Emails → adicione seu email (thiago.oliveira.comp@gmail.com)

6. Salve

Gratuito para até 50 usuários. Isso substitui definitivamente qualquer senha hardcoded no HTML.
