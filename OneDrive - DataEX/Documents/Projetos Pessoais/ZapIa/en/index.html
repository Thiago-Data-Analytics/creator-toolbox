Built with security
from the first byte

Your data and your customers' data are protected by enterprise-grade security protocols.

🔒
HTTPS + TLS 1.3
Public traffic is protected with HTTPS/TLS on all published environments. Additional protections depend on the active stack and enabled integrations.
🛡️
DDoS Protection + WAF
Protected by the Cloudflare network — the same infrastructure that protects banks and governments. Attacks are absorbed before reaching the server.

Claude models from Anthropic operate in MercaBot's backend with centralized operation control — simplifying use for businesses and reducing unnecessary browser exposure.
